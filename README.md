4. После завершения работы над домашним заданием сделайте коммит (`git commit -m "comment"`) и отправьте его на Github (`git push origin`).

### Задание 1

На лекции рассматривались режимы репликации master-slave, master-master, опишите их различия.

*Ответить в свободной форме.*

Ответ:
У master-slave есть один master сервер,когда у master-master их 2+ master сервера.
Если в m-s падает master,то необходимо выбрать другого master для дальнейшей работы.У m-m при падении одного master,другие смогут работать.
При использовании m-m может происходить конфликт или рассинхронизация данных из за одновременного изменения данных.
